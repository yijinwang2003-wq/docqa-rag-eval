**Key Takeaway**

Key takeaway: Query rewriting adds measurable value for underspecified, symptom-first, or diagnostic questions where the original query does not map cleanly to documentation structure (Q17: +71% relevancy). For well-specified questions, the current heuristic triggers unnecessarily, adding latency without retrieval benefit. A future improvement would replace the token-count and vague-reference heuristic with an LLM-based confidence signal that distinguishes genuinely ambiguous queries from precise ones that happen to contain pronouns.
